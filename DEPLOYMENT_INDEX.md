## üîß Configuration Files Reference

### Backend Configuration
- **`.env.backend.example`** - Environment variables for backend
  - Copy to `.env` for local development
  - Set values in Render dashboard for production

### Frontend Configuration
- **`frontend/.env.example`** - Environment variables for frontend
  - Copy to `frontend/.env.local` for local development
  - Set values in Netlify dashboard for production

### Deployment Configurations
- **`render-backend.yaml`** - Render deployment configuration
  - Reference for setting up Render service
  - Copy/paste values into Render dashboard
  
- **`frontend/netlify.toml`** - Netlify deployment configuration
  - Automatically picked up by Netlify
  - No manual setup needed
